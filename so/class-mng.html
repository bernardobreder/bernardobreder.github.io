Gerenciador de Classe

O gerenciador irá controlar novas classes instaladas, a sua descrição e especificação. O sistema operacional entende que cada aplicação misturado com a implementação do sistema possui um conjunto de classe versonado conhecido e gerenciado.

O sistema terá acesso a uma floresta de aplicações, incluindo o sistema, com suas descrições de classe organizada hierarquicamente na forma de pacote. Por exemplo, o sistema poderá ter instalado 3 aplicações, fazendo com que a floresta seja composta por 4 árvores, onde o primeiro corresponde ao do sistema operacional e as outras 3 das aplicações. Para as 4 cada raiz, uma hierarquia de pacote será visualizado com as classes específico para cada aplicação.
